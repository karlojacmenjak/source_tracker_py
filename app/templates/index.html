About Source Tracker

Source Tracker is a Discord bot that gathers real-time information from various Source Engine game servers using the Source Query (A2S) protocol. Whether it's player stats, server status, or game events, Source Tracker keeps your community informed.

Supported Games

  • Counter-Strike 2
  • Team Fortress 2
  • Garry's Mod
  • Left 4 Dead 2
  • And more...

Features

  • Real-time server tracking
  • Player statistics directly from the server
  • Customizable alerts for Discord servers
  • Easy Discord integration
  • Discord server commands
